Geração de valores automaticos para PK

A anotação @Id em um campo numerico pode ser precedida
da anotação @GeneratedValue(strategy=GenerationType.AUTO)
definido que o seu valore será incrementado em um a cada
novo registro.

ex:

@Id
@GeneratedValue(strategy=GenerationType.AUTO)
@Column(name="BID_ID")
public Long getBidID() {
	return bidID;
}
