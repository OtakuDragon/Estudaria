Eventos JavaFX

O tratamento de eventos padrões no JavaFX é feito por meio do 
registro de implementações da interface EventHandler<TipoEvento>,
através de métodos setOn.. presentes nos controles.

Apenas um event handler pode ser setado para cada tipo de evento padrão,
através dos métodos setOn.. .
-------------------------------------------------------------------------
ex:

Button button = new Button("Botão fodástico");

button.setOnAction(new EventHandler<ActionEvent>() {
	@Override
	public void handle(ActionEvent event) {
		System.out.println("do stuff");
		
	}
});

//Ou com lambda


button.setOnAction(event -> System.out.println("do stuff"));
