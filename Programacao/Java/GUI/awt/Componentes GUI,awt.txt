Classe TextField

Cria um campo de texto

criação de objeto

TextField txt = new TextField();

construtores

TextField(String arg, int arg);
// int representa tamanho maximo de caracteres do campo de texto e string um texto escrito dentro do campo de texto
//pode ter nenhum dos argumentos somente um ou os dois

metodos

txt.setEchoCharacter(char arg); converte o texto escrito naquele campo de texto no character definido usado para senhas
txt.getText(); retorna o texto do campo
txt.setText(); escreve texto no campo
txt.setEditable(boolean arg); true torna o campo editavel false não editavel é true por padrão
txt.isEditable(); retorna o q foi definido em setEditable();

As classes window

Os componentes da classe window são frame e dialog e se tratam de componentes que contem componentes assim como os 
panes só que em uma janela independente

Classe Frame

em portugues quadro é um componente de janela independente que pode conter outros componentes

Construtor

Frame janela = new Frame("String titulo"); //recebe um string para definir o titulo da nova janela ou não

Adicionar componentes

janela.add(componente);

Definir layout

janela.setLayout();

Definir tamanho da janela

janela.resize(int height, int width);

Mostrar ou não mostrar

janela.show(); //mostra a frame na tela
janela.hide();//esconde a frame
janela.isShowing();//retorna true se a janela estiver sendo mostrada e false se estiver escondida
janela.setFont();

Classe Dialog

Ou caixas de dialogo, são tambem janelas associadas a frames que podem conter componentes mas com menos recursos
são geralmente usadas para alertas e erros no programa, podem ser modais ou não modais sendo que modais
impedem os outros componentes do programa de ser executados até que a janela seja escondida

Construtores

Dialog Alerta = new Dialog(Frame arg,String titulo,boolean modal);

//Frame arg associa a caixa de dialogo a um frame existente
//String titulo funciona como o titulo de frame e tambem é opcional
//boolean modal = true nenhum componente da frame recebe foco ou açoes até que a caixa seja escondida

possue os mesmos metodos de customização de frame
