JSF

O Java server faces é a especificação para um framework
de componentes para desenvolvimento web em java

ele facilita a criação de interfaces para aplicações web

está especificação foi criada pela java community process

=========================================================
Implementações

é necessario uma implementação

a implemetação oficial da sun é chamada de Mojarra
