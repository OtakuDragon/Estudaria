Estrutura de um Script

Um script é um conjunto de codigo
que dá ação a um ou varios Game Objects
por meio do script é possivel controlar
todos os components de um Game object

Um script de unity é baseado em metodos
chamados pelo unity em certos momentos


==========================================

Variaveis

As variaveis em um script unity possuem
algumas diferenças de variaveis comuns javascript

1º - é possivel definir uma visibilidade para elas

- public - Variavel acessivel à o unity(ler 3º)

- static - Variavel acessivel à outros scripts por meio da chamada nomeDoScript.variavelEstatica

- private - Variavel apenas acessivel dentro do script

2º - é possivel definir um tipo de dado para elas

sintaxe:

visibilidade var nomeDaVariavel : tipoDeDado = Valor;//Visibilidade e Tipo de dado são opcionais

3º - Se uma variavel for publica e estiver dentro do escopo principal do script(fora de todos os metodos)
     ela será acessivel por meio do component script no game Object e será possivel mudar o seu valor atraves
    de um slider sem a necessidade de mudança no codigo
