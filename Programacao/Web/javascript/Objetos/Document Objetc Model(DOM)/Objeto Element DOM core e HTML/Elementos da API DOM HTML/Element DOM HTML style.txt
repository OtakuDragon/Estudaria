Element style

O elemento style representa o atributo HTML
style e possui como atributos os atributos
CSS

todos os elementos da API DOm HTML possuem
o elemento style dentro deles que é acessado
da seguinte forma:

element.style


=============================================
Atributo style para o codigo css da tag

style.cssText	//Define ou retorna todo o codigo html na tag ou seja valor do atributo style
