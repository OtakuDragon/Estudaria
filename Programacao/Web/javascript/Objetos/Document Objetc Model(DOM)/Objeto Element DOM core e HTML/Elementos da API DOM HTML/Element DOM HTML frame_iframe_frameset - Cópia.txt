Element frame/iframe

O elemento frame/iframe representa as tags html <frame> e <iframe> 
e possui como atributos todos os atributos e metodos de node alem dos
seus proprios atributos HTML
e um evento


=============================================
Atributos de frame/iframe

frame.align		//Sets or returns the value of the align attribute in an iframe	
frame.contentDocument	//Returns the document object generated by a frame/iframe	
frame.contentWindow	//Returns the window object generated by a frame/iframe	
frame.frameBorder	//Sets or returns the value of the frameborder attribute in a frame/iframe	
frame.height		//Sets or returns the value of the height attribute in an iframe	
frame.longDesc		//Sets or returns the value of the longdesc attribute in a frame/iframe
frame.marginHeight	//Sets or returns the value of the marginheight attribute in a frame/iframe	
frame.marginWidth	//Sets or returns the value of the marginwidth attribute in a frame/iframe	
frame.name		//Sets or returns the value of the name attribute in a frame/iframe	
frame.noResize		//Sets or returns the value of the noresize attribute in a frame	
frame.scrolling		//Sets or returns the value of the scrolling attribute in a frame/iframe	
frame.src		//Sets or returns the value of the src attribute in a frame/iframe	
frame.width		//Sets or returns the value of the width attribute in an iframe	

================================================
Eventos de frame/iframe

frame.onload	//Script to be run immediately after a frame/iframe is loaded

================================================
