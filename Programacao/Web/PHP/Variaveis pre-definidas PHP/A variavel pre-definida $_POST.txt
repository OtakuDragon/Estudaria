A Variavel predefinida $_POST

É usada para receber dados pelo method post de um formulario os dados vem em um array
no qual os indices são os nomes dos campos input e os valores os valores dos campos input

Codigo HTML da pagina

<form action="welcome.php" method="post">
Name: <input type="text" name="fname" />
Age: <input type="text" name="age" />
<input type="submit" />
</form> 

Codigo welcome.php acessando os dados

Welcome <?php echo $_POST["fname"]; ?>.<br />
You are <?php echo $_POST["age"]; ?> years old!

----------------------------------------------------------------


*O method post do <form> possui alguns detalhes

*Pode enviar diretamente informações confidenciais para o script

*Não tem limite na quantidade de dados que pode ser enviada
(o php define esse limite como 8 mb mais pode ser configurado em php.ini)

*Como as variaveis não são mostradas na URL não é possivel "Bookmark" a pagina

*Em elementos de formulario multipla escolha o nome do elemento tem que ser seguido de
 [] quando este elemento chegar ao array chegara como um array e podera ser utilizado como um
